Craigslist Redesign

UX/UI Design

Project

This project was an exercise in interaction design with a goal of redesigning the Craigslist iPad (768x1024) website into one that was easier to use and had better functionality. It was essential to utilize usability heuristics to create a consistent and elegant web experience. The current site has a lot of utility; however, it is locked behind a façade of ugly and garish UI. The goal of this project was not to alter the utility. Rather, the final design seeks to reveal that utility to the user by unlocking it from behind that façade.

Process

The process for this project followed the traditional interaction workflow, with each round of revisions being accompanied by usability testing and reflection. Task Flow diagrams were made based off user testing of the initial Craigslist site and helped inform the construction of initial paper wireframes. One key takeaway was that the site needed a consistent, easily-read top navigation bar. Additionally, styling was added to the various elements that would be reused across the site. The addition of gradient, dropshadow, and photographic elements alluded to neumorphism and helped add some level of physicality to the site.
