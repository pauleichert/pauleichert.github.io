Onitsuka Pop-Up Shop

The Onitsuka Pop-Up Shop project was an exercise in creating branding elements for a fictional pop-up shop for Japanese brand Onitsuka Tiger. The brand was renamed to Asics in the 1970's, though it was relaunched in One key priority with the project was creating a design system with typography, color palette, and design element that function as a whole. This process began with a long iterative design process that ultimately resulted in the final compositions which would be used to spread awareness for the event.

The process for this project took the form of both physical thumbnail sketches and quick digital sketches. The one key required element was the inclusion of the mantra of the historic brand—“anima sana in corpore sano”—a latin phrase meaning “a sound mind in a sound body.” Another required element was the use of a design element that encapsulated the brand’s deep history. Several design elements were considered in the iterative process, each one capitalizing on some aspect of the brand. The angled mountainous element alludes to the brand's Japanese past. I chose two color palettes for the project: "Mexico 68" and "Kill Bill." Each palette was designed after a significant touchstone in Onitsuka Tiger's brand history. The "Mexico 68" palette follows the colors of the Mexican flag, harkening back to the brand's involvement in the Mexico 1968 Olympics, at which the new, well-recognized Asics logo. The "Kill Bill" color palette alludes to the iconic yellow and black suit from both Uma Thurman's Kill Bill and Bruce Lee's Game of Death; both of which featured Onitsuka footwear in that same iconic colorway. Each palette touches on a different, but uniquely important, aspect of the Onitsuka brand's complex and rich history.
