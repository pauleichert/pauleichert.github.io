Personal Branding

Branding & Web Design

Project

This was a self-directed project that aimed to quickly and efficiently create a visual identity that could be used to build print or digital designs. The final identity needed to use two typefaces that worked together harmoniously. Additionally, a color guide needed to be made, so that this website could be built with a consistent style guide. The final result was the generation of some print materials and this portfolio website — the one you're reading right now.

Process

This project started with some casual exploration over the course of a few weeks. Considerations were primarily made for how the final website should look. As a result, the priority for the typography was for how it would look on the web. An icon was generated that could be used for print or digital materials, however; ultimately it was only used for the website's favicon. The website was initially developed with a loud, bright, maximalist style; however, users responded poorly to it and it was not very accessible, so the final design used a more minimalist style.
